Identity var (User Identification): window.ci360_identity (customer_id)

Ordem recomendada (pra bater com CI360):
1) Clique em Set Identity (isso salva no localStorage e faz ci360('identify')).
2) Se o CI360 de vocês usa Page View para “User Identification”, clique em Set Identity e depois recarregue a página (F5) pelo menos 1x.
3) Clique em Disparar evento (Click) e valide no Network (/t/s/).

Validação rápida: DevTools → Network → filtro /t/s/ → deve aparecer POST (200/204).
Obs: Reports/Diagnostics no CI360 têm delay (não é realtime). Use Network/Snowy para ver na hora.
